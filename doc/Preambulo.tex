% Resumo
\begin{center}
\textbf{RESUMO}
\end{center}
      \vspace{0.5cm}

\paragraph{}Este trabalho tem por objetivo apresentar uma nova técnica de medição de altura de ondas em beira de praia através de processamento digital de imagens. Os métodos de estimação atuais utilizam sensores montados em boias no mar, que não são práticos e necessitam de mão de obra especializada para serem instalados. Desta forma o método aqui proposto inova, possibilitando análises em larga escala e em tempo real, servindo inicialmente à população de surfistas. Este método realiza a análise sobre um vídeo e utiliza um algoritmo que avalia a variação da espuma formada pela arrebentação, detectando as ondas presentes no vídeo e estimando a sua altura. Um modelo geométrico da praia é utilizado para converter a altura estimada em \textit{pixels} para a altura real em metros. O método foi testado de duas maneiras: utilizando câmeras de \textit{smartphone} e um aparato de \textit{hardware} que foi construído para este projeto, com uma câmera menos potente. Ambos os equipamentos de captação de imagem produziram bons resultados em diferentes condições de mar e de clima. Assim, mostrou-se que o projeto é versátil, sendo capaz de trabalhar com fontes de dados diferentes, além de se provar robusto o suficiente para lidar com variações no ambiente de praia.

\paragraph{}

\noindent Palavras-Chave: processamento digital de imagem, visão computacional, oceanografia, estimação de altura de ondas marítimas, análise em tempo real.

% Abstract
\begin{center}
\textbf{ABSTRACT}
\end{center}
      \vspace{0.5cm}

\paragraph{}This project presents a new technique for nearshore wave height estimation through digital image processing. The current methods for wave height estimation require sensors embedded on sea buoys, which are not practical to use and require a specialist to be installed. Therefore, the proposed method innovates, it enables large scale real time analysis of sea wave data, initially providing information for the surfer population. This method analyses a video, and applies an algorithm which evaluates the variation of the foam level at the wave breakzone, detecting the waves captured by the video and estimating its height. A geometric model of the beach was used to convert the estimated height in pixels to the real world height in meters. Two data sources were used to test this method: smartphone cameras and a hardware apparatus built for this project using a lower grade camera. Both equipment produced satisfying results in different sea and climate conditions. Thus, it was shown that the project is versatile, being able to work with different data sources and proving itself robust enough to handle variances in the beach environment.


\paragraph{}

\noindent Key-words: digital image processing, computer vision, oceanography, sea waves estimation, real time analysis.
